## 1. Purpose

This process automates changing a customer's address by first verifying that the associated bank account is active, and only proceeding to update the account's status if that condition holds; it exists to prevent address changes being applied against accounts that are not in an active state.

Evidence: Lookup Account (action); Account Active? (decision); MFG Core Banking::Update Account Status (action)

## 6. Business rules

**BR-1.** Address changes require the account to be active before any update is performed.

Evidence: Note1 (note); Account Active? (decision)

**BR-2.** When the account is active, the account status is set to 'Updated' as part of processing the address change.

Evidence: MFG Core Banking::Update Account Status (action)
